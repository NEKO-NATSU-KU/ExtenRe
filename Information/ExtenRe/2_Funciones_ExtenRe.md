## ⚙️ How does ExtenRe work?

**ExtenRe** connects with projects that have been abandoned, are about to be abandoned, or simply need a deep update to keep working.

The process works as follows:

1. **A base project is identified** that needs to be reconstructed (for example, ReVanced Extended).
2. **The original repository is cloned** and its structure, dependencies, and code are analyzed.
3. **Structural and technical changes are applied**, such as:
   - API migration
   - Updating Java, Gradle, and other tool versions
   - Code modifications for compatibility with recent versions
4. **The reconstructed project (ExtenRe)** is published as a new repository, preserving the essence of the original but with improvements and updates that make it functional today.

In the specific case of ExtenRe, this has already been done with Inotia00's ReVanced Extended project, transforming it into an updated and functional version with Java 21 and Gradle 9.3.1.
