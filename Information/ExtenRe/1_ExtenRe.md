## 🧩 What is ExtenRe?

**ExtenRe** (Reconstructed Extension) is an open source project that originates from [ReVanced Extended by Inotia00](https://github.com/inotia00/Revanced_Extended).

The original project was forked and underwent multiple deep modifications to update, improve, and keep it alive. Among the most important changes are:

- **Complete migration** of the codebase
- **Update of obsolete or removed APIs**
- **Change from Java 11/17 to Java 21**
- **Gradle update**: from versions 7 and 8.14.3 to **Gradle 9.3.1**
- Modifications to the original code to ensure **compatibility with new versions** of dependencies and tools

ExtenRe is not just a copy, but an active **reconstruction** that seeks to give new life to projects that have been abandoned or need updating to remain functional.
