## ❓ What is ExtenVoid?

**ExtenVoid** is an open source project that acts as a **framework** for modifying and improving projects that are already in active development or maintenance.

Unlike ExtenRe, which works on abandoned projects to rebuild them from the ground up, ExtenVoid focuses on:

- **Cloned projects** that are currently working.
- **Specific modifications**: API changes, dependency updates, bug fixes, etc.
- **Continuous improvements**: Adding new functionalities without interrupting the original project flow.
- **Technological adaptations**: for example, migrating from one Java version to another, changing obsolete libraries to more recent ones, etc.

ExtenVoid does not seek to replace the original project, but to **extend its capabilities** through a set of practices and tools that facilitate controlled code modification.
