## 🔤 What does ExtenVoid mean?

The name **ExtenVoid** comes from the combination of two words:

- **Exten**: Short for "Extension" (same as in ExtenRe).
- **Void**: English word meaning "empty".

### Why "void"?

The term "void" has several meanings in programming and in the context of this project:

- In languages like Java or C, `void` indicates that a function returns nothing, but **performs actions**. It symbolizes the idea that ExtenVoid **acts** on the code without needing to create something from scratch, but rather modifying what exists.
- The "void" also represents the **space of opportunity**: an active project always has gaps to fill (improvements, fixes, new features) and ExtenVoid helps fill those spaces in an orderly manner.
- Metaphorically, it is like a "blank canvas" (empty) that is superimposed on the original project to draw on it without erasing what is already there.

Therefore, **ExtenVoid means "Void Extension"**, understood as the ability to extend an existing project by taking advantage of the spaces for improvement it offers.
