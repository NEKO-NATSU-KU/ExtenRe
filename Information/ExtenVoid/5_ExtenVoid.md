# 🌌 ExtenVoid

**ExtenVoid** is the complementary project to **ExtenRe**, designed to work with projects that are in an **active or maintenance state**.

While ExtenRe focuses on rescuing and reconstructing abandoned projects, ExtenVoid allows modifications, improvements, and changes to projects that are already functioning but need adaptations, updates, or new functionalities.

The name reflects its nature: "Void" represents the space of opportunity to add or modify code without breaking the existing structure. It's like a "blank canvas" within a living project, where developers can experiment and implement changes in a controlled manner.

ExtenVoid does not start from scratch, but rather **takes a cloned repository of an active project** and offers tools and patterns to apply modifications in an orderly, documented, and reversible way.
